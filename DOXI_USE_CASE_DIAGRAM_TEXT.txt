NOTES FOR DIAGRAM CREATION:
---------------------------
1. Use Draw.io (https://app.diagrams.net/) - Free and easy
2. Create 3 stick figures for actors (Patient, Doctor, Admin)
3. Create ovals for use cases
4. Use arrows to connect actors to use cases
5. Use dashed arrows for relationships between use cases
6. Group related use cases together
7. Use different colors for each actor's use cases
8. Add a "System" boundary box for system use cases

DIAGRAM STRUCTURE SUGGESTION:
------------------------------
Left Side: Patient Actor + Patient Use Cases
Center: System Use Cases (in a box)
Right Side: Doctor Actor + Doctor Use Cases
Top: Administrator Actor + Admin Use Cases

This creates a clear visual separation and makes the diagram easy to understand.
